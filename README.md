# ForgeFlow — Manufacturing Analytics 

ForgeFlow is a full end-to-end analytics and simulation ecosystem built to model 
a mid-sized Brazilian household-appliance manufacturer evaluating expansion into 
the United States. 

This project integrates:
- Real demand data (Olist)
- Real quality-signal data (SECOM)
- Synthetic manufacturing data (production orders, routing, BOM, plant capacity)
- Predictive modeling (demand forecasting & defect probability)
- Scenario simulation (future plant in the U.S.)
- Final Tableau dashboards for executives
